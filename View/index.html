T-Coin

This is a block chain built based off of Tyree Stevenson interpretation of block chain. The Blocks within this chain are called T Coins. T Coins can be mined and sent on this site.

Users must create a Wallet on this site to interact with T-Coins. This Wallet can then be used to store T-Coins, receive T-Coins, and send T-Coins

Mining page allows users to mine for T-Coins. To receive a T-Coin your computer must find a number bigger than than last T-Coin incrementer and divisible by 9.

Transaction page allows you to send the last
T-Coin in your Wallet to another users Wallet.
